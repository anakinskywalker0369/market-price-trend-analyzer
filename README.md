# Market Price Trend Analyzer

A data analysis tool designed to identify price fluctuations and trends in the technology market.

## 📈 Objective
To help users identify the best time to purchase by analyzing historical price data and filtering out inconsistent or erroneous entries.

## 🛠 Features
* **Data Cleaning:** Automatically handles missing values and invalid (negative) prices.
* **Trend Visualization:** Generates a professional line chart to track price changes over time.
* **Date Parsing:** Converts raw strings into structured time-series data for accurate analysis.

## 🚀 Built With
* Python
* Pandas
* Matplotlib

## 📋 Instructions
1. Upload your CSV file with `fecha` and `precio` columns.
2. Run `python CodigoAnalisis.py`.
3. View the generated trend chart.
